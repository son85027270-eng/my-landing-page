# 내 자기소개 랜딩페이지 PRD

## 1. 목적
32년 실무 경험의 경영지도사로 AI 기반 데이타와 전략을 바탕으로 3개월 안에 매출구조를 바꿔드립니다.

## 2. 사용자 (페르소나)
- 이 페이지를 '전달하는 사람(나)' :
  - 손영환/경영지도사/이노서스컨설팅 대표
  - 주로 어떤 상황에서 이 페이지를 공유하는가 : 업계 세미나 후 네트워킹 시간, 외부 강연 자리, 사내 타 부서 첫 미팅
- 이 페이지를 '받아 보는 사람(상대방)' : 중소기업 대표. 소상공인
  - 어떤 기기로 보는가 (스마트폰 / PC) : 스마트폰/PC
  - 얼마나 오래 볼 것 같은가 :
  - 무엇을 알고 싶어 하는가 :

## 3. 기능 (필수 / 선택)
### 필수 기능
1.페이지 맨 위에 '이름 · 직책 · 회사' 가 한 줄로 표시됩니다 (본문의 1.5배 크기).
2. 이름 바로 아래에 3~5줄 자기소개 문장이 표시됩니다.
3. 최근 경력 3건이 최신 순으로 카드 형태로 나열됩니다
   (연도 · 회사 · 직책 · 한 줄 요약).
4. 하단에 '이메일 보내기' 버튼이 있고, 누르면 기본 메일 앱이 열리며
   수신자 주소가 자동으로 채워집니다.
5. 하단에 '전화 걸기' 버튼이 있고, 누르면 전화번호가 클립보드에 복사됩니다.

### 선택 기능 (여유 있을 때)
1 페이지 상단에 원형 프로필 사진 한 장이 표시됩니다.
2. 경력 카드 우측에 회사 로고가 작게 표시됩니다.

## 4. 제약
### 의도적으로 하지 않는 것
-회원가입·로그인 기능은 넣지 않습니다.
- 방문자 통계·조회수·댓글·방명록은 넣지 않습니다.
- 다국어(영어·일본어 등)는 지원하지 않습니다. 한국어 단일로 만듭니다.

### 환경적 한계
- HTML·CSS만 사용합니다.
- PC와 스마트폰 모두에서 화면이 깨지지 않아야 합니다.
- 이미지는 프로필 사진 1장 이하로 제한합니다.

## 5. 성공기준 (완성 체크리스트)
- [ ]크롬 브라우저에서 페이지가 5초 이내에 열립니다.
- [ ] 스마트폰 세로 화면에서 모든 글자가 잘리지 않고 전체가 읽힙니다.
- [ ] 이름·직책·회사가 한 줄에 들어가 있습니다.
- [ ] 자기소개 문장이 3~5줄 범위 안에 있습니다.
- [ ] 경력 세 건이 최신 연도 순서로 정렬되어 있습니다.
- [ ] '이메일 보내기' 버튼이 메일 앱을 엽니다.
- [ ] '전화 걸기' 버튼이 전화번호를 클립보드에 복사합니다.
- [ ] 페이지에 색상이 네이비·흰색·짙은 회색 외의 색이 나타나지 않습니다.
- [ ] 이미지가 프로필 사진 1장 외에 추가로 나타나지 않습니다.
